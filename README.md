# Dubai-Rental-Market-Analysis-2020-2026

## Project Overview
This project analyzes Dubai rental market trends from 2020 to 2026 to identify factors influencing rental prices, property demand, and market patterns.

## Key Analysis
- Rental price trends over time
- Property type comparison
- Area-wise rental analysis
- Accessibility impact on rent
- COVID-19 market impact
- Rental price distribution
