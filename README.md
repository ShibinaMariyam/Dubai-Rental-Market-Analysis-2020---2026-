# Dubai-Rental-Market-Analysis-2020-2026
This project analyzes Dubai rental market trends from 2020 to 2026 to identify factors influencing rental prices, property demand, and market patterns.

## Dataset Source
The dataset used in this project was obtained from Kaggle: "Dubai Real Estate Sales and Rentals (2020–2026)" by Sergio Nefedov.
For this analysis, only the rentals.csv file was used, containing rental property data from January 2020 to April 2026. A subset of approximately 10,000 records was extracted from the original dataset for analysis.

Dataset Link: https://www.kaggle.com/datasets/sergionefedov/dubai-real-estate-sales-and-rentals-20202026

Dataset used in this project:UAE_Rental.csv

It includes information about:
- Property details (property type, size, bedrooms, bathrooms)
- Rental information (annual rent, rental trends)
- Location attributes (communities, zones, accessibility)
- Market-related factors influencing rental prices

The dataset was used to perform exploratory data analysis (EDA), identify rental price patterns, and understand the factors affecting Dubai's rental market.

## Key Analysis
- Rental price trends over time
- Property type comparison
- Area-wise rental analysis
- Accessibility impact on rent
- COVID-19 market impact
- Rental price distribution

## Conclusion

This analysis provides insights into the key factors influencing Dubai’s rental market from 2020 to 2026. The findings highlight how property characteristics, location, community, accessibility, and market trends impact rental prices.

The analysis shows that premium locations, larger properties, and high-demand communities generally command higher rental values. Accessibility, amenities, and property type also play an important role in determining rental demand and pricing.

These insights can help investors identify profitable opportunities, property owners optimize rental strategies, and tenants make informed decisions when choosing properties. Overall, the Dubai rental market demonstrates strong variation across locations and property segments, emphasizing the importance of data-driven decision-making in real estate analysis.
